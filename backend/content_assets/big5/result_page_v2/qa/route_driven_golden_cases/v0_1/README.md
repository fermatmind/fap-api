# Big Five V2 Route-Driven Golden Cases v0.1

This package is an advisory QA package for route-driven selector and pilot runtime readiness.

- Runtime use: `staging_only`
- Production use allowed: `false`
- Ready for pilot/runtime/production: `false`
- Body generation: none
- Content generation: none

The cases define route keys, profile-family coverage, expected registry refs, expected suppression refs, and metadata non-leak expectations. They are not report body assets and must not be connected to production governance.
