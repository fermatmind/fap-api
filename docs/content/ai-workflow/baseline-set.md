# AI 内容基准集（Baseline Set）规范

本文件定义 AI 内容基准集回归机制与发布前门禁要求。

## 基准集定义

- 基准集由 20 张 JSON 组成（覆盖核心内容类型）
- 每条基准集必须有固定 `id`/`slug`，且与任务包 8 schema 对齐
- 目录约定：`content_baselines/`（本闭环只写规范，不创建 JSON）

## 版本策略

- 基准集版本采用 `v1/v2/...` 方式递增
- 大改必须进行 baseline version bump（如结构变更、风格大幅切换）
- 仅限小改（错别字/措辞优化）可不 bump

## 回归口径

- 允许：错别字修复、措辞微调、不改变结构的微改
- 禁止：结构崩坏、字段缺失、风格大换血（除非 bump）

## 发布前门禁

- 必过：baseline gates + diff 不爆炸
- 脚本化回归后置到闭环 10D（本闭环不做脚本）
