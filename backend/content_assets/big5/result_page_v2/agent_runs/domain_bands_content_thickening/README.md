# Big Five Domain Bands Content Thickening Candidates v0.1

This backend-only package contains reviewed candidate assets for 5 OCEAN domains across 5 result bands. It is staging-only evidence for content asset review. It does not generate the final result contract, does not change runtime behavior, and does not allow production use.

Scope:
- 25 selector candidates
- 25 content candidates
- explanation, strengths, costs, common misreads, action guidance, and boundary language
- no frontend copy, CMS write, search change, rollout, or deploy

Source policy: existing FermatMind Big Five assets and the approved benchmark rubric are the working sources. 123test and Truity remain structure references only; no external wording or items are copied.
