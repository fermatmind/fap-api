# Copywriting No-Go List（文案禁区清单 · v0.2）

> 适用范围：Fermat Assessment Platform（费马测试平台）  
> 所有面对用户的文案：官网、小程序、分享图、广告投放、公众号文章等。

---

## 0. 使用方式

**原则：上线前，把重要文案与本清单对照一遍。**

- 只要出现「类似」风险表达——即使不是逐字一致——都视为命中禁区，需要修改或删除。
- 如果实在拿不准，就默认 **宁可保守一点**，用更中性的说法。

---

## 1. 医疗 / 诊断 / 治疗 相关禁区

### 1.1 严禁出现「诊断 / 疾病」表述

❌ 禁用例（包括同义改写）：

- 「本测试可以诊断你有没有抑郁症 / 焦虑症 / 双相障碍」
- 「测试结果显示，你已经属于重度抑郁」
- 「通过这份报告，我们可以判断你有心理疾病」
- 「你的精神状态已经出现病变 / 症状很严重」

✅ 推荐替代表达：

- 「本测试仅用于了解你的**性格倾向 / 情绪风格**，不构成任何医学或心理诊断。」
- 「如果你长期感到情绪低落、睡眠明显受影响，建议尽快与专业心理咨询师或医生沟通。」

### 1.2 严禁「治疗效果」承诺

❌ 禁用例：

- 「通过我们的测试 / 服务，可以治愈你的抑郁 / 焦虑」
- 「按我们说的做，你的心理问题就会好起来」
- 「保证你走出痛苦 / 拯救你的精神状态」

✅ 推荐替代表达：

- 「测试结果可以帮助你更好地认识自己的倾向，作为自我调整和寻求专业支持的参考。」
- 「如果这些困扰持续影响日常生活，建议在专业人士的帮助下进一步评估和处理。」

## 2. 自杀 / 自残 / 暴力 等高危内容

> 即使是「玩笑」「梗」，也不能在**官方结果页、广告、分享图**中出现。

### 2.1 自杀 / 自残

❌ 禁用例：

- 「这种人格经常想死 / 想消失」
- 「你可能已经不想活了」
- 「这类人一不开心就想自残」

✅ 如确有需要提到严重情绪，可用：

- 「如果你近期频繁出现‘活着没意义’、‘想消失’等念头，并且持续影响到你的生活，请尽快联系专业心理咨询师或前往正规医院就诊。」

### 2.2 对他人暴力 / 报复

❌ 禁用例：

- 「这种人格爆发时会毁掉一切 / 把所有人都搞死」
- 「你天生是复仇型人格，被惹到了就要让对方付出代价」

✅ 可以改为更中性的：

- 「在极端压力下，你可能会有较强的报复心或攻击冲动，这需要通过更健康的方式来表达情绪与设定边界。」

## 3. 歧视 / 羞辱 / 极端标签化

### 3.1 群体歧视

❌ 禁用例：

- 「XX 地区的人都……」
- 「XX 职业的人普遍很低级 / 没文化」
- 「这种人格根本没救，天生 loser」

包括但不限于：性别、性取向、地区、阶层、职业、学历、疾病等相关的贬低、嘲笑、侮辱。

✅ 改写方向：

- 聚焦「行为模式 / 倾向」，避免对人的价值做结论。
- 列出「优势 + 潜在盲区」，而不是“好人 / 坏人”。

### 3.2 人格价值排序 / 高低论

❌ 禁用例：

- 「INTJ 是最顶级人格，其他都一般」
- 「这种人格天生没前途」
- 「ENFP 就是长不大的小孩，不靠谱」

✅ 推荐表达：

- 「不同人格各自有适合的场景和优势，没有绝对好坏。」
- 「在某些工作 / 环境下，这类倾向更容易发挥优势。」

---

## 4. 绝对化命运论与人生决策绑架

### 4.1 「命中注定 / 一辈子都这样」类

❌ 禁用例：

- 「你注定找不到真爱」
- 「你这辈子都不适合当领导」
- 「你永远改变不了这种性格」

✅ 改写为：

- 「在当前倾向下，你在亲密关系中可能更容易遇到这些问题……」
- 「通过练习和环境调整，很多人格相关的行为模式是可以慢慢优化的。」

### 4.2 重大决策指令式话术

❌ 禁用例：

- 「根据测试结果，你应该立刻辞职 / 分手 / 退学」
- 「你必须和这种人格的人在一起才幸福」

✅ 安全表达：

- 「结果可以作为你思考职业 / 关系方向的一个参考维度，但不应成为唯一依据。」
- 「重大决定建议结合现实情况、他人反馈以及专业咨询等多方面信息。」

## 5. 未成年人相关内容

> 如果产品可能触达未成年人，需要格外谨慎。

❌ 禁用例：

- 「未成年人也可以自己做决定，不用和父母沟通」
- 「如果父母反对你，就完全不用理会」
- 「你比父母懂自己多了，不需要任何成年人的意见」

✅ 建议表达：

- 「如果你是未成年人，遇到困惑时，建议在信任的成年人（如父母、老师、学校心理老师）的陪伴下共同讨论。」
- 「在涉及转学、辍学等重要决定时，一定要与监护人和专业人士充分沟通。」

---

## 6. 数据隐私与个人信息展示

### 6.1 分享图 / 公共展示中的隐私

❌ 禁用例：

- 在分享图、海报中直接展示：
  - 真实姓名 + 学校 + 专业 + 班级 + 头像 + 具体班级号
  - 手机号、邮箱、微信号
  - 具体住址或工作单位的详细信息

✅ 推荐做法：

- 分享图只展示：
  - 人格类型 / 昵称（可选）
  - 部分关键词 / 成就徽章
  - 简单头像或抽象头像，不与真实身份强绑定（除非用户**明确同意**公开身份）。

### 6.2 暗示泄露隐私的文案

❌ 禁用例：

- 「只要转发，我们就能看到你所有测试记录」
- 「你做过的所有测试，我们都记得，一辈子不会消失」

✅ 建议透明说明：

- 「你的作答记录会在匿名 ID 下用于统计分析，帮助我们优化题目和报告，不会公开展示个人作答详情。」
- 若涉及登录账号，需额外说明数据用途和保留方式（可放在隐私政策中）。

## 7. 恐吓 / 焦虑营销话术

> 在心理、人格领域尤其容易踩坑。尽量避免用恐吓制造焦虑来促使用。

❌ 禁用例：

- 「不做这份测试，你就看不清自己，一辈子迷茫」
- 「现在不测，以后会后悔」
- 「99% 的人做完都震惊，你还不赶紧来？」

✅ 更健康的表达：

- 「如果你正在思考‘我到底是怎样的人’，这份测试可以提供一个有结构的参考视角。」
- 「测试结果不是标准答案，只是帮你多拿一个镜子看自己。」

---

## 8. 违法 / 违规 / 平台红线内容

> 同时遵守中国相关法律法规和各大平台（微信、公众号、广告平台等）的内容规范。

严禁涉及或宣传：

- 赌博、色情、毒品、非法集资等相关内容；
- 明示或暗示欺诈行为；
- 煽动仇恨、分裂、极端主义等内容；
- 违反微信广告 / 公众号内容规范的表达（如夸大功效、使用「国家级」「最高」「唯一」等绝对化词汇）。

## 9. 常见「危险句式」与推荐改写

以下是日常很容易写出来、但**需要警惕**的一些句式。

### 9.1 「你就是……」定性句式

❌ 例子：

- 「你就是不靠谱的人」
- 「你就是优柔寡断」
- 「你就是社交废物」

✅ 推荐改写：

- 「在面对选择时，你可能更容易犹豫，担心做错决定。」
- 「在社交场合，你可能会感觉有些拘谨，需要一定时间才放松下来。」

### 9.2 「你一定 / 必然 / 必须」句式

❌ 例子：

- 「你一定会在感情中受伤」
- 「你必须改变自己，否则永远不会幸福」

✅ 推荐改写：

- 「如果不留意这些倾向，你在亲密关系中可能更容易受到伤害。」
- 「有意识地练习边界感和沟通，可以帮助你在关系中更好地保护自己。」

### 9.3 「过度夸大人格潜能」句式

❌ 例子：

- 「INTJ 只要稍微努力就是人上人」
- 「ENFP 天生是贵族人格」

✅ 推荐改写：

- 「在许多创新型环境中，这种人格的优势更容易被看见。」
- 「如果能结合自律和长期规划，你的创意和热情有机会转化为很实际的成果。」

## 10. 上线前自检 Checklist（快速对照）

在你写完一篇结果页文案 / 推广文案 / 分享图文案后，至少问自己以下问题：

1. 有没有出现「诊断 / 疾病 / 治疗」相关词语或暗示？
2. 有没有鼓励、轻描淡写或浪漫化自杀 / 自残 / 暴力的内容？
3. 有没有把某类人群 / 人格说成「低级 / 没救 / 天生失败」？
4. 有没有用人格结果绑架重大人生决定（让用户立刻辞职、分手等）？
5. 分享图 / 公共展示处是否泄露了用户的真实隐私信息？
6. 是否用了恐吓式、焦虑型营销话术？
7. 是否存在与法律法规或平台规则明显冲突的内容？
8. 如果一个情绪敏感的用户看到这段文案，会不会感觉被羞辱或被攻击？

**只要有一个问题答「是」或「不确定」，就应该重写或收紧表达。**
