# Big Five V2 Real-user Rollout Dry-run v0.1

This package records dry-run governance evidence for a future Big Five V2 real-user production rollout.

It does not enable production rollout, expose real users, or mark the release ready for production.

Dry-run scope:
- allowlisted rollout simulation.
- percentage rollout simulation.
- rollback simulation.
- blast radius simulation.
- fail-closed simulation.
- incident halt simulation.
